# ZipReader

Hosted at: _________

Displays words in a text one-by-one rather than page-by-age.
This technique called Rapid Serial Visual Presentation (RSVP) can more than **double** your reading speed!

## Local Setup
Start by cloning the repo and navigating to the frontend directory. Then run the following commands.

1. ```npm install```
2. ```npm start```

...and boom! The website should show up in your browser.

## Credits
